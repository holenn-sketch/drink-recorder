# 饮酒记录器

这是一个纯前端网页版饮酒记录器，不依赖后端、网络、npm 或第三方库。界面采用 Apple 风格的流体云背景、玻璃层级、大圆角、柔和高光和真实按压反馈。

## 使用方式

直接用浏览器打开 `index.html` 即可运行。

也可以部署到 GitHub Pages，当前版本是静态网页模式。多人实时同步和微信授权登录需要额外后端/API，不能只靠 GitHub Pages 完成。

数据会保存在当前浏览器的 `localStorage` 中：

- `drink_people_records`
- `drink_quick_amounts`

## 功能

- 添加任意数量参与者
- 按快捷酒量增加或减少杯数
- 自定义、编辑、删除、恢复默认快捷酒量
- 单人清零、删除
- 全部清零、清空全部
- 总人数、总杯数、最高酒量统计
- 添加顺序 / 酒量从高到低排序

杯数内部使用 `quarters` 计量，`1 quarter = 0.25 杯`，避免浮点数累积误差。
